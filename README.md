About This Project

Rocket AI Assistant is a Python-based virtual assistant created using Flask.
It can take text input from the user and respond like a smart assistant.

The assistant can:

Tell current time and date

Answer questions using Wikipedia

Open websites like Google, Instagram, Facebook, Snapchat

Respond to greetings

Tell who created it

Speak responses using text-to-speech

This project is useful for learning:

Flask backend

Python automation

Voice assistant logic

API handling

Web integration

Features

Web-based assistant using Flask

Text-to-speech using pyttsx3

Wikipedia search support

Open popular websites

Shows current date and time

How the Program Works

User enters a command in the web interface

Flask sends the request to backend

The command is processed

Assistant gives a reply

If required, it opens websites or fetches information

Response is returned to the user

Example Commands

What is the time

What is today’s date

Open Google

Open Instagram

Who created you

Search Wikipedia for Python

Exit

How to Run the Project

Install required libraries:

pip install flask pyttsx3 speechrecognition wikipedia


Run the program:

python app.py


Open browser and go to:

http://127.0.0.1:5000/
